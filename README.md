# Rock&Roll UpsideDown

Content infrastructure for the live music industry.

Capture. Archive. License. Distribute. At the speed the industry actually operates.

## Quick Start

```bash
pnpm install
cp .env.example .env.local
docker compose up -d
pnpm dev
```

Portal: http://localhost:3100
